# Introduction

This is my vim configuration

# Requirements

- Vim with `terminal` feature
- Vim with `python` feature
- [Nerd Fonts](https://www.nerdfonts.com/) (for vim icons)

# Installation

```bash
cd ~
git clone git@github.com:Lesords/vim-config.git ~/.vim
```
